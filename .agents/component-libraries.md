# Reference: Free React & Tailwind Component Libraries

This document lists the recommended websites for free React, Tailwind CSS, and shadcn/ui components, blocks, and hooks to use when expanding or polishing the Everonix Technologies client interface.

---

## 1. Animated & Premium Visuals (Framer Motion & Tailwind)
*   **[Magic UI](https://magicui.design):** Offers 150+ free animated components designed specifically for design engineers. Great for landing pages, text effects, bento grids, and interactive elements.
*   **[Aceternity UI](https://ui.aceternity.dev):** Famous for stunning, modern motion-rich blocks (hero grids, card hover effects, glowing lines, parallax sections).
*   **[Cult UI](https://www.cult-ui.com):** A repository of custom-designed, animated components and minimal layouts.

## 2. Shadcn/ui & Radix-based Blocks
*   **[21st.dev](https://21st.dev):** The "npm for design engineers". A massive community marketplace containing thousands of shadcn-compatible custom inputs, hooks, buttons, and sections.
*   **[Shadcnblocks](https://shadcnblocks.com):** Provides a large library of free, copy-pasteable blocks (headers, features, CTA strips, dashboards) that fit directly into shadcn.
*   **[Origin UI](https://originui.com):** A sleek library focused on custom shadcn element refinements (dropdowns, toggles, badges).

## 3. Comprehensive Open-Source Tailwind Libraries
*   **[Flowbite](https://flowbite.com):** One of the largest open-source component ecosystems. Great for standard UI elements like Datepickers, Accordions, and Tabs.
*   **[Preline UI](https://preline.co):** A massive open-source library featuring production-grade B2B SaaS sections, dashboard layouts, and forms.
*   **[Meraki UI](https://merakiui.com):** Focuses on simple, clean, responsive, and ready-to-use tailwind blocks (pricing tables, FAQ columns, cards).
*   **[DaisyUI](https://daisyui.com):** A Tailwind CSS plugin that gives you bootstrap-like semantic classes (e.g. `btn`, `card`) to keep code sizes small.
