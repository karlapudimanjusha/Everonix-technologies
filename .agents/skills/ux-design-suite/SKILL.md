## 🛠️ Combined Quick Checklist

* **Refactoring UI:** Design in grayscale first. Use defined spacing (4/8/16/24/32/48px). Design with text contrast, not just size.
* **UX Heuristics:** Keep system status visible. Ensure errors are easy to recover from. Prevent errors before button click.
* **Hooked UX:** Define the trigger → action → reward → investment loop. Keep user friction low.
* **iOS HIG:** Minimum touch targets of 44x44px. Respect device safe-areas.
* **Design Sprint:** Prototype fast with static data, then validate with screenshots or UAT testing.
